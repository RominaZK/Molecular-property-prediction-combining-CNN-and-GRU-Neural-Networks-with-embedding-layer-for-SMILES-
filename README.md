# Molecular Property Prediction using CNN and GRU

## Project Synopsis
This project focuses on predicting molecular properties from SMILES representations using deep learning techniques. 
The objective is to demonstrate how neural networks can be applied to complex, structured scientific data within a data analytics and machine learning context.

## Dataset
- Molecular datasets represented as SMILES strings

## Tools & Technologies
- Python
- TensorFlow / Keras
- Convolutional Neural Networks (CNN)
- Gated Recurrent Units (GRU)
- Data preprocessing and model evaluation

## Methodology
- Data preprocessing and encoding of SMILES strings
- Feature representation using embedding layers
- Design and training of a hybrid CNN + GRU deep learning model
- Model evaluation and performance analysis

## Results & Insights
The model achieved strong predictive performance, highlighting the effectiveness of deep learning architectures for molecular property prediction tasks.

## Skills Demonstrated
- Deep learning model development
- Handling structured and complex datasets
- Feature engineering and data preprocessing
- Model evaluation and interpretation
